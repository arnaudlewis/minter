# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in minter, please report it responsibly.

**Email:** Send a description of the vulnerability to the maintainer via [GitHub private vulnerability reporting](https://github.com/arnaudlewis/minter/security/advisories/new).

**Please include:**
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

**Do not** open a public issue for security vulnerabilities.

## Response

You can expect an initial response within 72 hours. We will work with you to understand the issue and coordinate a fix before any public disclosure.
